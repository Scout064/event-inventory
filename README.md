# event-inventory
"Event Inventory" is a very lightweight Flask App to manage your Event Inventory (subtle, I know ;))

## ⚠️ Important Notice

This repository provides the **core open-source code and documentation** for the Event Inventory system.

It is intended for developers who are willing to **invest time in setup, configuration, and debugging**.  
Out-of-the-box execution is **not guaranteed** and may require manual adjustments depending on your environment.

If you are looking for a **ready-to-use, easily deployable solution**, please use the official Docker image instead:  
👉 https://github.com/Scout064/event-inventory-licensed/pkgs/container/event-inventory-licensed
